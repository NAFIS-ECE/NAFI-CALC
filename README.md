A simple calculator built using Python.

- Addition
- Subtraction
- Multiplication
- Division

PROCESS OF RUNNING:
## How to Run

1. Clone this repository
   git clone https://github.com/NAFIS-ECE/NAFI-CALC.git
2. Run the script
   python NAFI-CALC.py
3. Enter two numbers and an operator (+, -, *, /) when prompted
